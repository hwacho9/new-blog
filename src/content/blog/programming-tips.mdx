# 프로그래밍 초보자를 위한 5가지 팁

프로그래밍을 처음 시작하는 것은 쉽지 않습니다. 새로운 언어, 개념, 도구들을 배우는 과정에서 많은 어려움을 겪을 수 있습니다. 이 글에서는 프로그래밍 초보자들이 더 효과적으로 학습할 수 있는 5가지 팁을 소개합니다.

## 1. 작은 프로젝트부터 시작하기

처음부터 큰 프로젝트를 시작하면 쉽게 지치고 포기할 수 있습니다. 대신, 작은 프로젝트부터 시작해 점진적으로 규모를 키워나가는 것이 좋습니다.

```python
# 간단한 계산기 프로그램
def calculator():
    num1 = float(input("첫 번째 숫자를 입력하세요: "))
    op = input("연산자를 입력하세요 (+, -, *, /): ")
    num2 = float(input("두 번째 숫자를 입력하세요: "))

    if op == '+':
        return num1 + num2
    elif op == '-':
        return num1 - num2
    elif op == '*':
        return num1 * num2
    elif op == '/':
        return num1 / num2
    else:
        return "잘못된 연산자입니다."

print(calculator())
```
